🌿 Paradise Nursery

A simple React + Redux shopping app for browsing and purchasing houseplants.

🚀 Features
Landing page with company intro and background image
Product listing with categorized houseplants
Add to cart functionality
Dynamic cart icon (updates item count)
Shopping cart with:
Quantity increase/decrease
Total cost calculation
Delete items
Checkout (Coming Soon)
🛠️ Tech Stack
React
Redux Toolkit
Vite
CSS
📂 Pages
Home (Landing Page)
Plants (Product Listing)
Cart (Shopping Cart)
