# 🌿 Paradise Nursery

A simple **React + Redux** shopping application for browsing and purchasing houseplants.

---

## 🚀 Features

- 🌄 Landing page with company intro and background image  
- 🌱 Product listing with categorized houseplants  
- 🛒 Add to cart functionality  
- 🔢 Dynamic cart icon (updates item count)  
- 🧾 Shopping cart with:
  - ➕ Increase / ➖ Decrease quantity  
  - 💰 Total cost calculation  
  - ❌ Remove items  
  - 🚧 Checkout (*Coming Soon*)  

---

## 🛠️ Tech Stack

- ⚛️ React  
- 🧠 Redux Toolkit  
- ⚡ Vite  
- 🎨 CSS
